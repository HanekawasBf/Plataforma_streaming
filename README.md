- Anime Lat 

Página web para aficionados al anime donde puedes explorar un catálogo de series y ver episodios directamente desde el sitio.


--- ¿Qué incluye?

- Página de inicio con una sección destacada y tarjetas de los animes disponibles
- Páginas de reproducción individuales para cada anime, con un reproductor de video y selector de episodios
- Barra de búsqueda en el navbar para encontrar contenido rápidamente
- Series recomendadas en el sidebar de cada página de reproducción
- Diseño responsive que se adapta a celular, tablet y escritorio
